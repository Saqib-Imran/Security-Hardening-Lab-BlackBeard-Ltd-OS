# OS Optimisation & Security Hardening Project – BlackBeard Ltd (Case Study)

## 🛡️ Project Overview
This repository documents a cybersecurity optimisation project completed as part of a university module. The scenario focused on enhancing the performance and security posture of Windows 11 in a virtualised enterprise environment for a fictional company, **BlackBeard Ltd**.

## 🎯 Objectives
- Perform comparative analysis of DOS, Windows, and Linux.
- Optimise and secure Windows 11 using best practices.
- Demonstrate system changes through benchmarking and screenshots.
- Deliver findings via a professional report and presentation.

## 🖥️ Technologies & Tools
- **Oracle VirtualBox** (virtualisation)
- **Windows 11 Pro (VM)**
- **Task Scheduler, Group Policy Editor, PowerShell**
- **Disk Management, Windows Defender, BitLocker**

## 📂 Contents
- `/report` → Full written comparative analysis + presentation PDF
- `/screenshots` → Evidence of system optimisation and security configuration
- `/benchmark_results` → Pre and post-optimisation benchmarks
- `/appendix` → Logs and extended system settings used during the setup

## 🧠 Key Outcomes
- Reduced system startup time
- Hardened firewall, AV, and log retention settings
- Secure user and group access configuration
- Automated backup and full-system scanning
- Strategic recommendations for wider deployment

## 🔐 Note
For licensing reasons, no Windows installation files or virtual disks are included.
